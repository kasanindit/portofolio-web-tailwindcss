StaySense is an Android App uses machine learning-based predictive solution to predict potential customer churn in telecommunication companies using XGBoost Algorithm.

RasaNusa

RasaNusa is an Android-based application to recognize food using image scanning and machine learning technology. The app is designed to raise awareness of local culinary heritage, helping users find out more information about the food they encounter.

Project Dicoding Story App

Dicoding Story App is a mobile application that allows users to view a list of stories shared by others as well as their own. Users can create and upload stories by either selecting images from the gallery or capturing photos directly using CameraX. The app also includes a location feature to display the user's location when submitting a story.

Project Dicoding Event App

Dicoding Event App is a mobile application that displays a list of events from Dicoding. It features three main sections: upcoming events, favorite events, and past events. The app fetches data from the official Dicoding API, allowing users to explore and manage events efficiently.
